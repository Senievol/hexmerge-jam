## Hex Madness

![Hex Madness](screenshots/screenshot001.png "Hex Madness")

### Description

Place and merge towers to protect your base against enemies!
This game is very unfinished, sadly we didn't have enough free time this week to do everything we wanted to.

### Features

 - Place towers on the grid
 - Merge towers by placing them next to each other
 - Upgrade your base to defend against enemies
